# plot_branch_node

ニューラルネットワークにおいてノードの次数にどれぐらい偏りがあるのかを調査

例えば, linearにおける隠れ層ではノードの次数は以下のようになる

隠れ層ノードの数はその層への入力数と同じで, 隠れ層ノードの次数はその層への出力数と同じ
